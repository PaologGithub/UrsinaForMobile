## How to Contribute

1. **Fork the repository.**
2. **Create a new branch:**
   ```bash
   git checkout -b feature/your-feature-name
   ```
3. **Commit your changes:**
   ```bash
   git commit -m "Description of your changes"
   ```
4. **Push your branch and open a pull request.**
